# 更新日志

## 2026-06-06

### 新增
- **MQTT Discovery 数据推送**：支持通过 MQTT 自动在 HA 中创建实体，无需手动配置 template。配置 `MQTT_HOST` 即可启用，详见 [MQTT 使用指南](MQTT.md)
- **PostgreSQL 数据库支持**：`DB_TYPE=postgresql` 可连接 PostgreSQL，支持 `PG_*` / `POSTGRES_*` 环境变量命名
- Web 控制台浏览器标签页 favicon（闪电图标）

### 优化
- **月用电量图表**：改为柱状图（用电量）+ 折线图（电费）混合展示，双 Y 轴独立刻度
- **余额时间显示**：从 `2026-06-06T00:00:00` 简化为 `2026-06-06`，只显示日期
- **应交金额**：为 0 时显示绿色 "已交清" 提示
- **上月账单 / 年度显示**：格式优化为 `用电量 / 电费` 的紧凑展示

## 2026-06-02

### 新增
- Web 控制台面板集成指南（[HA_PANEL.md](HA_PANEL.md)、[QUICK_PANEL.md](QUICK_PANEL.md)）
- 支持通过 HA Webpage 仪表盘功能嵌入 Web 控制台

### 修复
- `panel_iframe` 集成在 HA 2024.4+ 已废弃，文档更新为 Webpage 仪表盘方式
- HTTPS 混合内容限制说明和 Nginx 反向代理解决方案

## 2026-05-21

### 新增
- **阶梯用电数据采集**：住宅用户自动获取一/二/三阶梯已用、剩余及当前阶段
- **电费余额增强信息**：应交金额传感器 `sensor.prepay_balance_xxxx`
- **Web 控制台**：浏览器查看运行日志、户号图表、手动触发同步
- **企业微信推送**：同步完成后推送多户用电汇总
- **大模型验证码识别**：支持火山引擎豆包大模型（`CAPTCHA_SOLVER=llm`）
- 统一数据库设计（6 张表），SQLite / MySQL 双引擎

### 优化
- 用户名自动从国网获取，无需手动配置
- 密码登录失败自动切换二维码登录兜底
- 忽略户号仅跳过抓取，保留数据库历史数据
